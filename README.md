💰 Personal Budget Tracker (Full Stack)

Welcome to the Personal Budget Tracker — a fully functional, visually appealing budgeting application built using the powerful MERN stack (MongoDB, Express.js, React.js, and Node.js).
This app empowers users to take control of their finances by offering a seamless way to track, manage, and visualize expenses. With secure user authentication, users can create their own accounts, log in securely, and manage personal financial data privately.

-Whether it's keeping tabs on daily spending or planning monthly budgets, this app makes budgeting easy with:
-Simple & intuitive expense form for adding items
-Predefined categories like Food, Shopping, Entertainment, Education, etc.
-Date-wise filters to view expenses by any specific day or month
-Instant insights into spending habits using dynamic pie charts
-Options to edit or delete expenses anytime
-Secure JWT-based authentication for safe login and registration
-Responsive UI with a soft pastel theme and minimalistic design

It's not just a tracker — it's your financial dashboard, always accessible and easy to use!
